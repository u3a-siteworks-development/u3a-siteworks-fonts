The fonts are produced under the SIL Open Source Licence. See https://openfontlicense.org/.
The updated fonts here are u3a Sans Light, Light Italic, Regular, Italic, Bold and Bold Italic.

The files in the zip file 'u3a Sns 30 Dec 2023.zip' are the .WOFF2 and .OTF versions.
The .WOFF2 files are used in the WordPress theme fonts folder.
The .OTF files are used in any word processor.

The 'u3a Sans Fontforge.zip' file contains the FontForge database files, which should be used as the basis of any improvements, corrections, or additions.

The fonts are produced under the SIL Open Source Licence. See https://openfontlicense.org/.
The updated fonts here are u3a Sans Light, Light Italic, Regular, Italic, Bold and Bold Italic.

There are three types of file for each font version:
The .SFD files are Fontforge database files, and should be used as the basis of amendments and improvements.
The .WOFF2 files are used in the WordPress theme fonts folder.
The .OTF are OpenType files and are used in any word processor.
